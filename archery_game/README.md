A fun, bright, and cartoony 3D arcade archery game with intuitive drag-to-shoot controls!

## Features

- **Cartoony 3D Graphics**: Bright, colorful, stylized visuals with flat shading
- **Simple Controls**: Drag on the trackpad/mouse to aim and release to shoot
- **Fun Environment**: Cartoony clouds, trees, and colorful targets
- **Physics**: Realistic arrow flight with gravity
- **Scoring**: Hit the colorful targets to earn points
- **Visual Effects**: Bouncy animations and colorful feedback

## Getting Started

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

The game will automatically open in your browser at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## How to Play

1. **Aim**: Click and drag on the screen to aim your arrow
2. **Power**: The longer you drag, the more power you'll have (watch the colorful power bar!)
3. **Shoot**: Release to fire the arrow
4. **Score**: Hit the colorful targets to earn points (100 points per target)

## Controls

- **Mouse/Trackpad**: Click and drag to aim, release to shoot
- **Touch**: Works on touch devices too!
