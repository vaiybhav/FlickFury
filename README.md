# Basketball

![Alt text](assets/images/demo_pic.png?raw=true)

[Basketball][basketball]

A remake of Facebook Messenger's basketball game! This game is a frontend browser game built with JavaScript, Phaser, and Canvas.

## Features
- Ball is shot in a direction based on the cursor's movement
- Ball changes size to make experience 3D like
- Sound effects make experience like REAL LIFE!
- Ball collides with and bounces on/off rim
- Motion tweens of happy or sad emojis appear based on success of the shot
- Besides the first shot, ball spawns in random locations

[basketball]: http://bonbonlemon.github.io/basketball/

# 3D Minigolf Game

A clean 3D portrait-oriented minigolf game with single player and multiplayer modes. Control the ball with a simple trackpad/mouse drag gesture.

## Features

- **3D Graphics**: Built with Three.js for smooth 3D rendering
- **Physics Engine**: Cannon.js for realistic ball physics
- **Drag-to-Shoot**: Simple drag gesture to aim and shoot the ball
- **Game Modes**: 
  - Single Player: Play solo and try to complete the hole in as few strokes as possible
  - Multiplayer: Take turns with another player
- **Portrait Orientation**: Optimized for portrait mode on mobile devices
- **Obstacles**: Navigate around obstacles to reach the hole
- **Score Tracking**: Track strokes for each player

## How to Play

1. Select a game mode (Single Player or Multiplayer)
2. Drag on the screen to aim and set power
3. Release to shoot the ball
4. Try to get the ball in the hole in as few strokes as possible
5. In multiplayer mode, players take turns automatically

## Controls

- **Mouse/Trackpad**: Click and drag to aim, release to shoot
- **Touch**: Touch and drag on mobile devices

## Technologies

- Three.js (3D graphics)
- Cannon.js (Physics engine)
- Vanilla JavaScript
- HTML5/CSS3

## Setup

Simply open `index.html` in a modern web browser. No build process required.

## Browser Compatibility

Works best in modern browsers that support:
- WebGL
- ES6 JavaScript
- Touch events (for mobile)
